## Prop `bp` Use prop `bp` (breakpoint) for control [responsive offcanvas](https://getbootstrap.com/docs/5.2/components/offcanvas/#responsive) which hide content outside the viewport from a specified breakpoint and down. Above that breakpoint, the contents within will behave as usual. For example, `bp="lg"` hides content in an offcanvas below the `lg` breakpoint, but shows the content above the `lg` breakpoint. ```html Launch default offcanvas ... ... ... ... ... ``` Resize your browser to show the responsive offcanvas toggle.
