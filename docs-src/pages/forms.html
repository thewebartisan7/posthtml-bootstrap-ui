### With label and help We can set the props with `aware` once in the `x-form.group` component like `id` or `name` and then we don't need to set most of the others props. ```html First and last name Please provide your first and last name ``` First and last name Please provide your first and last name
### Form layouts The `x-form.group` component can be used also to create form layouts of previous example, and pass props to label, input and help via aware.
