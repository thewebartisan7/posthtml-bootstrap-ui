## Dropdown menu as `ul` list By default, the dropdown use a `div` tag for `.dropdown-menu`. In case you want a `ul` tag, use the prop `list`. For avoid to add to each dropdown item, you can add once to `x-dropdown.menu` by using `aware:list`. It can be just `aware:list` or `aware:list="true"` for use list and `aware:list="false"` for use `div`. ```html Toggle dropdown First item Second item Third item ``` Toggle dropdown First item Second item Third item
## Dropdown item as link, button or text By default the dropdown item use link, if you want to use a button instead, then use `x-dropdown.item.button`.
